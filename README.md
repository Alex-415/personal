## Projects

### Microservice Health Simulator
Production-grade distributed system simulator with real-time health monitoring, failure propagation, and circuit breaker patterns. Simulates 5-10 interconnected microservices with async Python backend, tracks latency/error rates, and visualizes service dependencies with React dashboard. WebSocket streaming for live metrics.

Built with Python 3.11, FastAPI, asyncio, React, TypeScript, WebSockets, Docker, deployed on Render.com + Netlify

### Network Inspection Platform
Production-grade TCP port scanner with async Python backend. Features concurrent scanning with semaphore-based rate limiting, SSRF protection, DNS validation, and structured logging. Clean architecture suitable for telecom-grade review.

Built with Python 3.11, asyncio, FastAPI, deployed on Render.com
